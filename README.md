# hackster

hackster is a news app using the Hacker News REST API. You can think of hackster as the little brother of Hacker News

## Usage

### Installation

Install the dependencies

```sh
$ yarn install
```

### Serve

Start the server

```sh
$ yarn start
```

Access from your browser:

```
localhost:3000
```

### Test

```sh
$ yarn test
```

### Demo

https://hackster-fir.herokuapp.com/
