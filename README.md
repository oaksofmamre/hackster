# hackster

hackster is a news app using the Hacker News REST API. You can think of hackster as the little brother of Hacker News

## Usage

### Installation

Install the dependencies

```sh
$ npm install
```

### Serve

Start the server

```sh
$ npm start
```

Access from your browser:

```
localhost:3000
```
